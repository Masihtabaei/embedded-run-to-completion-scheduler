void check_password ( )

Function used for checking an entered password.

Comparing the entered password and taking one of the following decisions based on the result of comparison:

  • Entered password is correct -> Waiting for 0,5s and restarting again (+ updating the lcd)
  • Entered password is wrong:
    • We have more chances -> Reducing the number of chances, updating the lcd and continuing the program
    • No more chances left -> Updating the lcd and stoping the program (it happens later in an another function)
